## Getting Started
```bash
npm install
npm run dev
npm run build
npm run lint
```

then go to `http://localhost:3000` to preview the site.

## Project Structure
```
app/
├─ layout.tsx        # Global HTML scaffold + fonts
├─ page.tsx          # Landing page content & animations
├─ globals.css       # Tailwind imports + custom animation utilities
components/
├─ Logo.tsx       
public/
├─ kiosk-display.svg 
```
